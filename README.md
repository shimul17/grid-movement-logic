# ♟️ Grid Movement Logic

A Python-based simulation of a pawn moving randomly on a 10x10 grid. This project demonstrates core **Object-Oriented Programming (OOP)** principles like Abstraction, Inheritance, and Encapsulation.

## 🚀 Key Features
- **Boundary Logic:** The Pawn never goes outside the 10x10 grid (0-9 range).
- **Level-Up System:** Unlocks diagonal movement after initial steps.
- **Victory Condition:** The Pawn searches for a 'Treasure' at position **(9, 9)**.
- **Path Tracking:** Keeps a full record of the journey.

## 🛠️ Concepts Used
- **Abstract Base Classes (ABC)**
- **Inheritance & Method Overriding**
- **Randomization & Coordinate Math**
